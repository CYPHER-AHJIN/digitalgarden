### Common Attack Patterns ‚Äî Quick Recognition

**Brute Force:** Many 4625 from same IP against same account ‚Üí check for 4624 success

**Password Spray:** Many 4625 from same IP against MANY accounts, 1-3 attempts each ‚Üí check for 4624

**Phishing Chain:** Email received ‚Üí User opens ‚Üí Office spawns shell process (Sysmon 1) ‚Üí Network connection (Sysmon 3) ‚Üí File dropped (Sysmon 11)

**C2 Beaconing:** Regular HTTP/DNS requests at fixed intervals ‚Üí low byte variance ‚Üí to newly registered domain

**Lateral Movement:** 4624 Type 3 from workstation to workstation ‚Üí 5140 Admin$ access ‚Üí New service on target (7045)

**Ransomware:** vssadmin delete shadows ‚Üí Mass file modifications ‚Üí Ransom note creation ‚Üí All in rapid succession

**Credential Dumping:** Sysmon 10 (lsass access) with specific GrantedAccess values ‚Üí Followed by 4624 Type 3 from many hosts (PTH)
